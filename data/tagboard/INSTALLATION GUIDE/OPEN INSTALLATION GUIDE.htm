Installation 1)  Variables to edit in config.php

It is not necessary to edit config.php until you have got your tagboard working however the variables that can be edited in config.php each have comments by them explaining their purpose.


2)  Upload

Upload all the files required (see above) into a directory preferably called "tagboard" so your link to tag.php would be www.yourdomain.com/tagboard/tag.php


3)  Editing your htmlcode.dat

If you have chosen to use a directory called "tagboard" or "board" and you are going to have your tagboard on your homepage then you have to edit a few things in your htmlcode.dat file (edits shown in red):

<link rel="stylesheet" href="tagboard/stylesheet.php" type="text/css">

and

<iframe name="tag" border="0" frameborder="0" width="100%" height="200" align="center" src="tagboard/display.php" target="_blank"></iframe>

and

<form method="POST" action="tagboard/tag.php" target="tag" name="tagboard">

and finally

<input type="button" class="xtra" value="Xtras" OnClick="tagboard/popUp('insert.php')">

5.  Administering Your Tag Board

To administer your Tag Board all you have to do is go to www.yoursite.com/tagboard/admin.php and log in with the username and password you defined in config.php.  Once in you will see options in a left hand side menu.

Administering Tags - this option allows you to edit and delete tags from your tag board.
   Editing - to edit a tag, click edit and modify the post in the box provided, you can only edit the message
   Deleting - to delete click the delete button and confirm the pop up.
IP Banning - this option allows you to ban unwanted posters, to find a posters IP, click on the name
Support - if your still stuck, this link will take you to the forums
About - about the CJ Tag Board
Logout - this will log you out of the admin
